# 图片批量添加水印工具

一个纯前端的图片批量水印工具，无需安装任何软件，直接在浏览器中打开即可使用。所有操作均在本地完成，图片不会上传到任何服务器，保障隐私安全。

## ✨ 功能特性

- **批量处理**：支持同时上传多张图片，一键添加水印
- **自定义水印文字**：输入任意文字作为水印内容
- **丰富的样式配置**：
  - 文字大小（像素）
  - 旋转角度
  - 字体选择（Arial、Times New Roman、Courier New、Verdana）
  - 透明度（0-100）
  - 水平间距与行间距
  - 字体颜色
- **图片预览**：点击图片可放大预览，支持左右箭头和键盘方向键切换
- **配置导入/导出**：将水印配置保存为 JSON 文件，方便下次复用
- **多种下载方式**：
  - 逐张下载
  - 打包为 ZIP 一键下载

## 🚀 使用方法

1. 使用浏览器打开 `wtmk.html` 文件
2. 点击「选择文件」上传一张或多张图片
3. 在输入框中填写水印文字
4. 根据需要调整水印样式参数（大小、角度、透明度等）
5. 点击「添加水印」按钮
6. 预览效果后，点击「下载全部」或「打包下载」保存图片

## ⚙️ 配置说明

| 参数     | 默认值   | 说明                   |
| -------- | -------- | ---------------------- |
| 文字大小 | 50 px    | 水印文字的像素大小     |
| 旋转角度 | -30°     | 水印文字的旋转角度     |
| 字体     | Arial    | 水印文字使用的字体     |
| 透明度   | 30       | 水印透明度，范围 0-100 |
| 水平间距 | 200 px   | 水印之间的水平间距     |
| 行间距   | 200 px   | 水印之间的垂直间距     |
| 字体颜色 | #FFFFFF  | 水印文字颜色           |

可以通过「导出配置」按钮将当前参数保存为 JSON 文件，下次使用时通过「导入配置」按钮快速恢复。

## 🛠️ 技术栈

- 原生 HTML / CSS / JavaScript（无需构建工具）
- [JSZip](https://stuk.github.io/jszip/) — 生成 ZIP 压缩包
- [FileSaver.js](https://github.com/eligrey/FileSaver.js/) — 触发浏览器文件下载
